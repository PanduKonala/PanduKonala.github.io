Overview

This project is funded by Amrita Vishwa Vidyapeetham. The objective is to design and develop a Cyber-Physical System for the safe transport system of Liquid Petroleum Gas (LPG) using ultra-sonic, MQ9 & DHT11 sensors which are linked to the cloud.

Problem Statement

Liquid petroleum gas is a highly flammable material that has to be contained very securely. To monitor the containment effectively, IoT provides a reasonable solution for real-time safety monitoring.

Hardware Components

  • DHT11: Temperature sensor to detect if liquid has reached its flashpoint
  • MQ9: Gas sensor for detecting LPG, CO, CH4 leakages
  • Ultrasonic Sensor: Measures distance between container lid and liquid level
  • NodeMCU: Open-source IoT platform with ESP8266 Wi-Fi SoC

Software Platforms

  • Arduino IDE: For programming and uploading code to boards
  • Adafruit Cloud: Cloud service for storing and retrieving sensor data
  • IFTTT: For creating automated alerts and notifications

Key Features

  • Real-time monitoring of temperature, gas levels, and liquid levels
  • Automated alerts to authorities when safety thresholds are exceeded
  • Cloud-based data storage and analysis
  • Automatic suppression measures activation

Preamble

The current world of the internet is mainly focused on centralized approaches. The majority of these services have complete control over the user who utilises their products. However, this approach is faster in delivering content to the user but lacks in the security and privacy of users data. To address this issue we have proposed a framework that safeguards users security and privacy through a decentralised approach with little or no cost. The following specification particularly describes the invention and how it is to be performed.

Claims

  • A framework for low-cost decentralized approach for media streaming using open source protocol
  • Utilizes bare minimum infrastructure to provide high security and privacy for the users. Thus making it resource effective
  • Implementation is targeted at users who value privacy, security and uncensored content

Key Innovation

This patent introduces a novel framework leveraging the InterPlanetary File System (IPFS) protocol to enable decentralized media streaming. Unlike traditional centralized streaming services, this approach prioritizes user privacy and data security while maintaining cost-effectiveness through peer-to-peer architecture.
